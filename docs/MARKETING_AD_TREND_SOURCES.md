# Marketing Advertising Trend Monitoring Sources for Codex

- Created: 2026-05-18
- Timezone: Asia/Seoul
- Language: ko-KR

## Goal

마케팅 광고 업계 최신 트렌드, 광고 소재 레퍼런스, 매체 업데이트, 커뮤니티 반응을 정기적으로 수집하고 AE/제안서/소재기획에 활용 가능한 요약본으로 변환한다.

## Recommended Monitoring Order

1. 매체 공식 업데이트: 광고 운영에 바로 영향을 주는 공지 확인
2. 소재 확인용 사이트: 경쟁사 소재, 후킹 문구, 썸네일, CTA 확인
3. 국내 마케팅·광고 트렌드: 한국 시장 실무 이슈 확인
4. 소비자·키워드 트렌드: 수요 변화, 시즌성, 연관어 확인
5. 커뮤니티·실무 반응: 운영자들이 실제로 겪는 문제 확인
6. 글로벌 광고·마케팅 뉴스: 큰 흐름과 다음 트렌드 확인

## Priority Rules

- **A**: 매일 또는 주 1회 반드시 확인. 실무 영향도가 높고 광고 운영/소재/제안서에 바로 반영 가능.
- **B**: 주 1회 확인. 인사이트 보강, 소재 레퍼런스, 제안서 근거 자료로 활용.
- **C**: 월 1회 확인. 국내 실무 직접 영향은 낮지만 글로벌 흐름 참고용.

## Daily Clip Output Schema

Codex는 매일 수집 결과를 아래 필드로 저장한다.

- date
- source_name
- category
- title_or_topic
- url
- summary_3_lines
- why_it_matters_for_performance_marketing
- suggested_action
- related_media_platform
- creative_angle
- proposal_usefulness_score_1_to_5

## Weekly Report Output Schema

Codex는 주 1회 아래 항목을 기준으로 요약 리포트를 생성한다.

- top_10_trends
- platform_updates
- winning_creative_patterns
- consumer_keyword_changes
- community_pain_points
- recommended_experiments
- proposal_insertable_insights

## Suggested Codex Prompt

```text
You are a marketing advertising trend monitoring agent for a Korean performance marketing AE.

Your task:
1. Visit the listed sources by priority and cadence.
2. Collect new information about advertising trends, creative examples, platform updates, consumer keyword trends, and community discussions.
3. Do not simply summarize articles. Convert each finding into practical implications for a performance marketing agency.
4. For creative references, extract:
   - hook
   - target audience
   - pain point
   - offer
   - CTA
   - visual structure
   - funnel stage
5. For platform updates, extract:
   - changed feature
   - affected advertisers
   - risk
   - opportunity
   - recommended action
6. For community posts, extract:
   - recurring problems
   - real practitioner sentiment
   - tactical solutions
7. Save daily results as `daily_marketing_trend_clip_YYYY-MM-DD.md`.
8. Save weekly results as `weekly_marketing_trend_report_YYYY-WW.md`.

Output language: Korean.
Tone: practical, direct, agency AE-friendly.
Do not include low-value generic summaries.
Prioritize actionable insights for Naver, Kakao Moment, Google Ads, Meta Ads, TikTok Ads, ecommerce, local business, beauty/clinic, finance, B2B lead generation, and smart store advertisers.
```
